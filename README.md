# Manzanas: The Next Apple

##  Live Demo

**Deployed at:** [dbms-manzanas.ddns.net](https://dbms-manzanas.ddns.net)

> **Note:** This is an educational project. We do not own, sell, or represent any real products or brands displayed on this website.

## Project Overview

Manzanas is a fully functional e-commerce web application developed as the final submission for the **Database Management System** course. The platform specializes in selling premium electronics including:
- Smartphones
- Laptops
- Desktop Computers

## Features

- Complete e-commerce functionality
- Admin dashboard for product management
- User-friendly interface
- Responsive design

## Academic Context

This project demonstrates practical implementation of database management concepts including:

- Database design and normalization
- CRUD operations
- SQL queries and optimization
- Frontend-backend integration
- Data persistence and management

### Technologies Used

- **Backend:** PHP and MySQL
- **Frontend:** HTML5, CSS3, JavaScript, TailwindCSS
- **Database:** MySQL
- **Hosting:** Google Cloud Platform
